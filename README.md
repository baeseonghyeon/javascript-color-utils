# javascript-color-utils

## Features

### `MaterialColor` (Material Color Generator)

usage

```javascript
const palette = new MaterialColor('#ffffff')

// palette result
{
    50 : '#......'
    100 : '#......'
    200 : '#......',
    ...
    500 : '#ffffff',
    900 : '#......',
    A100 : '#......',
    A200 : '#......',
    A400 : '#......',
    A700 : '#......',
}
```
